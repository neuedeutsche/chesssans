# ChessSans

Public distribution assets for the Chess.com brand font, ChessSans.

## Contents
- `ChessSansVariableFont.ttf`
- `Format — OTF/` (OpenType)
- `Format — TTF/` (TrueType)
- `Format — WOFF/` (Web Open Font Format)
- `Format — WOFF2/` (Web Open Font Format 2)
- `Rounded — OTF/` (Rounded OpenType)

## Quick Use
- Desktop and design workflows: `ChessSansVariableFont.ttf`
- Web: prefer `Format — WOFF2/`, fall back to `Format — WOFF/`

## Origin (Summary)
ChessSans began as a customization of the open-source Montserrat typeface and evolved into a bespoke font system tailored to Chess.com. The goal was to align the typeface with the Chess.com logo’s open, vertical character and improve legibility in compact UI contexts.

Key changes include:
- Open counters to increase clarity at small sizes and improve character distinction.
- Narrower proportions (about 20% narrower than Montserrat) to fit UI layouts and longer words.
- A broader design space with weight and width options to adapt to UI, branding, and responsive layouts.

## Design Principles (Summary)
- Open counters for clarity and readable internal spaces.
- Proportions optimized for UI and longer text.
- Readability prioritized for timers, buttons, and compact interface elements.
- Weight and width axes for flexibility across branding and product UI.
